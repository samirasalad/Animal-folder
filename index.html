The website features photographs of animals from pixabay.com. The key to animal photography is patienence. It can take a long time to get the right light an the best veiw of the animal. when photographing animals you do not have a lot of control you need to wait for them.

Web Design Exercise

Animal Photography is a website exercise on image galleries available on WebDevStudent.com. The images used in this exercise are royalty-free photos from pixabay.com. JavaScript was used to create the photo gallery, and Bootstrap columns were used for responsive layout.
